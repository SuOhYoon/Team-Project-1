<플랜 A>
1. 회원가입
-ID, PW
-회원가입시 정보를 member.db에 저장
2. 로그인
-ID, PW
-ID와 PW를 member.db에서 맞는지 확인
-맞으면 member.db에서 정보를 클라이언트로 전송
3. 자동차 구매
3-1. 필터 (상세 검색)
-브랜드, 연식, 키로수, 자동차명, 사고/무사고 여부, 차종, 가격대
-필터에서 체크한 항목을 car.db로 전송해서 추출->사용자에게 전송
3-2. 검색 결과
-정렬기능(가격순, 연식순, 주행거리순)
3-3. 선택 차량 보기
-차량이미지, 가격, 연식, 키로수, 자동차명, 사고/무사고 여부, 차종, 가격대, 딜러, 옵션(간단하게), 판매하는 딜러의 
-업로드(판매자 전용)
4. 후기 게시판(커뮤니티)
-글 작성, 수정, 삭제 기능
-댓글 기능
-좋아요, 싫어요 기능
-글을 작성하면 작성한 글 정보(작성자, 게시글 이름, 내용)를 board.db에 저장
5. 관리자 페이지
-관리자 권한 부여 -> 체크사항 member.db에 전송 후 member.db에서 요청사항 처리
-사용자 가입/탈퇴 승인 -> 체크사항 member.db에 전송 후 member.db에서 요청사항 처리
-게시글 삭제 / 수정 기능 -> 체크사항 board.db에 전송 후 board.db에서 요청사항 처리
-------
1. 사용자 회원가입
회원가입 -> 정보 입력 후 확인 -> member.db에 새로운 회원정보 추가 -> 가입 완료
2. 사용자 로그인
로그인 -> 정보 입력 -> member.db에서 사용자,id,pw가 일치하는지 확인 -> 일치한다면 로그인 성공
3. 차량 구매
상세 차량 정보 입력 -> car.db에서 관련 정보로 필터링된 차량 리스트 나열 후 추출 -> 정보 보여주기
3-1. 상세 창
보려는 차량 클릭 -> car.db에서 관련 정보 추출 -> 정보 보여주기
4. 구매 후기 게시판
4-1. 글 열람
보려는 글 클릭 -> board.db에서 관련 정보 추출 -> 글 보여주기
4-2. 글 작성
작성 버튼 클릭 -> member.db에서 로그인 정보 확인 -> 로그인 되어 있으면 작성 페이지로 이동 -> 글 작성 -> 작성된 글 board.db에 추가
-> 글 작성 완료
4-3. 글 수정
수정 버튼 클릭 -> member.db에서 로그인 정보 확인 및 작성자 본인이 맞는지 확인 -> 맞다면 수정 페이지로 이동 -> 글 수정 -> 수정된 글 board.db에서 업데이트
-> 글 수정 완료
4-4. 글 삭제
삭제 버튼 클릭 -> member.db에서 로그인 정보 확인 및 작성자 본인이 맞는지 확인 -> 맞다면 삭제 경고창 표시 -> 클릭하면 board.db에서 해당 게시글 삭제
-> 글 삭제 완료
5. 관리자 페이지
5-1. 관리자 페이지 접속
로그인 -> member.db에서 사용자,id,pw가 일치하는지 확인하고 관리자 권한이 있는지 확인 -> 일치하고 관리자 권한이 있다면 '내 정보' 창에 '관리 기능' 추가 -> '관리 기능'
누르면 관리자 페이지 접속 -> 접속 전 다시한번 member.db에서 관리자 권한이 있는지 확인 -> 있다면 관리자 페이지 로드
5-2. 관리자 권한 부여
관리자 페이지에서 '권한 부여' 클릭 -> member.db에서 전체 사용자 추출 -> 특정 이용자 '권한 부여'에 체크 -> member.db에서 해당 이용자 관리자 권한 추가 -> 완료
5-3. 게시글 수정
-> 일반적인 글 삭제 / 수정 기능이랑 비슷하나 관리자는 사용자 일치 여부와 상관없이 모든 글 삭제.수정 가능
5-4. 회원 관리 기능
관리자 페이지에서 '회원 관리' 클릭 -> member.db에서 전체 사용자 추출 -> 특정 이용자 체크 후 '회원정보 삭제' 클릭 -> member.db에서 해당 회원정보 삭제 -> 완료
